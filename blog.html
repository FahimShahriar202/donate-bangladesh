Important FAQ

What is DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, making it possible for programming languages (like JavaScript) to interact with and manipulate the content, structure, and style of the document.


How do you select an element from DOM?

By using:- document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName()


what is event delegation in the context of DOM, why is it useful.

Event delegation is a technique in the DOM where you add a single event listener to a parent element instead of multiple event listeners to individual child elements. The event bubbles up from the child to the parent, allowing the parent to handle events for all its child elements. It is useful for better performance and cleaner code


How do you manipulate element's attribute and styles using DOM?

By using getAttribute(attributeName), setAttribute(attributeName, value), removeAttribute(attributeName)
